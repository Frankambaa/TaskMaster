🔄 Persistent Chat History Demo

✨ New Feature: Persistent Chat History

The chatbot now remembers your conversations across page reloads and different browser sessions! This means:

  • 🔄 Chat history persists when you refresh the page
  • 🌐 Works across different browsers and devices (for identified users)
  • ⚙️ Configurable number of messages to remember (default: 10)
  • 🔐 Secure and private - only your messages are restored

📋 How to Test:

  1. Open the chat widget below
  2. Send a few messages to the bot
  3. Refresh this page (F5 or Ctrl+R)
  4. Open the chat widget again - your messages should still be there!

🔧 Configuration Options:

You can customize how many messages to persist:

ChatWidget.init({
    apiUrl: 'http://localhost:5000',
    user_id: 'demo_user_123',
    username: 'Demo User',
    email: 'demo@example.com',
    device_id: 'web_browser',
    persistentHistoryCount: 15  // Keep last 15 messages
});

🎯 Technical Details:

  • Backend Endpoint: /widget_history - Retrieves conversation history
  • Storage: Database-backed with user identification
  • Security: User input sanitization and message limits
  • Performance: Efficient loading of recent messages only
