🎤 ElevenLabs Button (Purple Icon)

Click this button in the chat widget to start ElevenLabs voice mode

📋 How to Use ElevenLabs Voice Agent

  1. Open Chat Widget: Click the chat button in bottom corner
  2. Start ElevenLabs: Click the 🎤 button (not the 🎙️ button)
  3. Listen to Welcome: AI will greet you with premium voice
  4. Speak Naturally: Ask questions or make requests
  5. Use Voice Commands: Say "stop" to pause responses
  6. Continuous Chat: System listens automatically after each response
  7. Disconnect: Click 🎤 button again to stop
🔑 API Key Required: ElevenLabs integration requires ELEVENLABS_API_KEY environment variable to be set. Without it, you'll see an error message.

🔄 Voice Processing Flow

  • 🎤 Browser speech recognition captures your voice
  • 📝 Text is sent to RAG/AI tool system (same as typing)
  • 🧠 AI processes through knowledge base or API tools
  • 🎵 Response is synthesized with ElevenLabs premium voice
  • 🔊 High-quality audio plays automatically
  • 🔁 System continues listening for next question

🎛️ Voice Controls

  • 🎙️ Regular Voice: Uses gTTS/Kokoro with call-style interface
  • 🎤 ElevenLabs Voice: Premium AI voice with purple interface
  • Stop Commands: Both systems respond to "stop", "pause", "wait"
  • Independent Systems: Can switch between modes safely
💡 Tip: Use ElevenLabs mode for the highest quality voice experience. The regular voice mode (🎙️) is still available for comparison or as a fallback option.
